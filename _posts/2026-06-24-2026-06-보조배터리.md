<h3 style="font-size:17px;margin:18px 0 4px;">Q. 20000mAh면 스마트폰을 몇 번이나 충전하나요?</h3>
<p style="margin:0 0 12px;">A. 기종과 배터리 용량에 따라 다르지만, 일반적인 스마트폰 기준 대략 3~4회 정도로 보면 됩니다. 충전 과정에서 손실이 있어 표기 용량을 그대로 다 쓰지는 못합니다.</p>
<h3 style="font-size:17px;margin:18px 0 4px;">Q. PD 22.5W 고속충전이 꼭 필요한가요?</h3>
<p style="margin:0 0 12px;">A. 급하게 짧은 시간에 충전해야 하는 경우가 많다면 유용합니다. 다만 폰과 케이블도 고속충전을 지원해야 제 속도가 나오니 함께 확인하세요.</p>
<h3 style="font-size:17px;margin:18px 0 4px;">Q. 케이블 일체형과 분리형 중 뭐가 좋나요?</h3>
<p style="margin:0 0 12px;">A. 일체형은 케이블을 따로 안 챙겨도 돼 휴대가 편하고, 분리형은 케이블 교체나 다른 기기 연결이 자유롭습니다. 매일 가볍게 들고 다닌다면 일체형이 편합니다.</p>
<h3 style="font-size:17px;margin:18px 0 4px;">Q. 보조배터리를 비행기에 가지고 탈 수 있나요?</h3>
<p style="margin:0 0 12px;">A. 보조배터리는 위탁수하물이 아닌 기내 반입이 원칙이며, 항공사·국가별로 용량(Wh) 기준 제한이 있으니 탑승 전 해당 항공사 규정을 확인하는 것이 안전합니다.</p>
